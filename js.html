Introdução ao JavaScript

JavaScript é a linguagem de programação mais popular do mundo!

O que é JavaScript?

  • JavaScript é a linguagem de programação da Web.
  • ECMA-262 é o nome oficial do padrão. ECMAScript é o nome oficial da linguagem.
  • Com o JavaScript podemos fornecer interatividade em nossos sites.
  • Grandes empresas como o Google, YouTube, Netflix, Amazon e IBM usam JavaScript em seus sites.

Por que estudar JavaScript?

JavaScript é uma das 3 linguagens que todos os desenvolvedores web devem aprender:

  • HTML para definir o conteúdo das páginas web
  • CSS para especificar o layout das páginas da web
  • JavaScript para programar o comportamento das páginas web

Imagine uma página web como se fosse um corpo. O HTML que é onde fica todo o nosso conteúdo é o esqueleto do nosso site. O CSS que é usado para estilizar nossa página é a aparência física do nosso corpo, como é visto. E o JavaScript é o nosso cérebro, que faz tudo funcionar!
